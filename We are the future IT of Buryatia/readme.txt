Требования к системе: ОС Windows 10/11 (Тестирование на других ОС не проводились)

Инструкция по запуску сервера: 

Запуск сервера:

Из папки distr запускаем server.exe
Указываем порт подключения (любой)
Указываем имена пользователя(ей) которым разрешен доступ (Если нексколько, указываем через пробел, возможно ошибки если у пользователя имя через пробел, если поле останеться пустым то вход разрешен всем пользователям)
Выбираем из списка "Минимальную версию Windows" для подключения к серверу
Нажимаем "Запустить сервер"

Запуск клиента:

Из папки distr запускаем client.exe
В строку "IP адрес сервера" вводим IP сервера (IP можно посмотреть в окне запущенного сервера)
Указываем порт (который указали на сервере)
Нажимаем "Подключиться к серверу"
Использование системы: При подключении клиента к серверу, автоматически происходит проверка минимальной версии windows у клиента и сверяется с заданной на сервере. Если версия windows у клиента соответствует или выше версии указанной на сервере, сервер оставляет клиента подключенным к себе. Если версия windows у клиента ниже выбранной минимальной версии на сервере, то соединение автоматически разрывается. Помимо этого одновременно происходит проверка соответствия введенного имени пользователя на сервере с именем пользователя клиента. В случае неверного ввода имени пользователя на сервере, соединение автоматически разрывается. Для удобства реализована кнопка "Очистить логи". Так же в папке с сервером автоматически создается файл log.txt со всеми подключениями (кнопка "Очистить логи" на него не распространяется).
