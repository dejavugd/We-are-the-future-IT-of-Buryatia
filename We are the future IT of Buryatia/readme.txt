Добрый день уважаемые члены жюри и организаторы мероприятия. Наша команда «БРМТИТ» завершила работу над проектом клиент серверного приложения «Система Проверки Клиентов» (СПК)
Целью проекта было создание клиент серверного приложения для проверки политик безопасности у пользователя.
Для достижения цели были выполнены следующие задачи:

1.  Переделана и улучшена графическая часть сервера и клиента. 
2. Добавлен список стандартных  портов подключения клиента с возможностью ввода порта вручную. 
3. Реализована проверка клиента по «Имени пользователя» При пустом значении сервер пропускает всех пользователей. Либо можно перечислить несколько пользователей через пробел. Только для них будет доступ.
4. Реализована проверка на «Минимальную версию ОС» При несоответствии условий проверки, доступ для клиента блокируется.
5. Добавлены кнопки «Запуск» и «Остановка» сервера и клиента
6. Добавлена кнопка «Очистить логи»
7. Добавлена функция “Запрещенные программы”, в котором можно ввести имя приложения *.exe (notepad.exe). Возможно добавление нескольких приложений через пробел. Если на стороне клиента будут запущены приложения из данного списка, соединение к серверу разрывается.
8. Добавлено окно “Подключенные пользователи” в котором можно отслеживать состояние подключений пользователей и отключать их в случае необходимости. ( функция автообновления пользователей пока не успели реализовать )
9. Добавлена функция проверки загруженности процессор у клиента. При достижении 80% загрузки процессора у клиента, происходит отключение от сервера. ( В коде проверка прописана каждые 10 с. ) Но в графическом виде реализовать ее пока не успели.
10. В окне «Логирования сообщений» выводятся правила установленные сервером “Минимальная версия ОС”, “Разрешенные пользователи”, “Запрещенные программы.
11. Все проверки были перенесены с сервера на сторону клиента.

Так же планировали реализовать проверку на установленный антивирус и логирование в отдельный файл в виде таблицы.
Но, к сожалению, не успели по времени.

В целом считаем что с поставленной задачей справились.
